# 会议记录

本页面用于集中整理TATEN 工作室的会议记录，包括项目讨论、阶段总结与内部决策内容，作为团队协作与回溯的重要参考。

---

## 说明

- 所有会议记录请统一存放在本目录下  
- 文件命名格式：`YYYY-MM-DD.md`
- 在 `config.mts` 的 `text` 的命名格式：`YYYY-MM-DD 会议主题`
- 每次会议结束后24 小时内整理并提交
- 未公开内容请标注为【内部】

---
