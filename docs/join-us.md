# 加入 TATEN

如果你对技术充满热忱，渴望在实战中磨炼技能，并愿意与一群志同道合的伙伴共同构建有影响力的开源项目，TATEN 欢迎你的加入！

## 我们能带给你什么？
- **真实项目实战**：参与 [TATEN-OJ](/projects/taten-oj) 等核心项目的研发与维护。
- **技术氛围**：扁平化沟通，定期技术分享，拒绝无效内卷。
- **极客工具栈**：体验 GitHub 全流程协作、自动化 CI/CD 以及专业的文档规范。
- **共同成长**：从代码审查（Code Review）中获得快速反馈，提升工程能力。

### 后端开发
- 熟悉 Go / Node.js / Python 其中之一。
- 了解基础的数据库设计 (MySQL/Redis) 及 RESTful API 规范。
- *加分项：了解 Docker 容器化或微服务架构。*

### 前端开发
- 熟练使用 Vue 3 / React 及 TypeScript。
- 对 UI/UX 有追求，能还原高质量的设计稿。
- *加分项：有 VitePress 或 Tailwind CSS 使用经验。*

### 文档与运营
- 负责技术文档的润色与组织，管理社区交流。
- 具备良好的文字功底，能将复杂的技术逻辑讲清楚。

## 申请流程

1. **提交申请**：点击下方按钮填写申请表单或提交 GitHub Issue。
2. **初步沟通**：我们会邀请你进行一次简单的线上技术/意向交流。
3. **试用期**：根据情况可能有一个限时一周的 Mini Task。
4. **正式入队**：获得团队权限，开启你的 TATEN 之旅！

::: tip 如何申请
目前我们支持以下两种方式（任选其一）：
- [在腾讯问卷填写申请问卷](https://wj.qq.com/s2/25766356/765a/)
- [在 GitHub 提交申请 (推荐)](https://github.com/TATENcn/repo/issues/new)
:::

## FAQ

::: details Q: 我技术比较菜，可以加入吗？
**A:** 只要你具备基础知识并有极强的**自主学习能力**，我们非常欢迎“潜力股”。
在 TATEN，我们更看重你解决问题的思维，而不仅仅是当下的工具熟练度。
:::

::: details Q: 加入后会有压力吗？
**A:** 我们是基于**兴趣和使命驱动**的团队，协作相对自由。
我们不强制打卡，但希望你对领取的任务**有始有终**。如果你遇到了困难，及时在群里沟通比闷头死磕更重要。
:::
