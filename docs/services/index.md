# TATEN公用云服务使用说明

团队成员CN059在2026年2月14日购置一台三年期的 8C 8G 30Mbps 香港云服务器，将该服务器作为团队共用服务器。同日，CN059将sudo权限共享给小林和小xin，并部署了 frps 服务。小xin也部署了较为完整的 kubernetes 服务。次日，CN059决定撰写TATEN云服务使用文档，方便团队内部成员方便快捷地接入TATEN云服务平台
